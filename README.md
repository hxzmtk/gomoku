# 五子棋
> 在线五子棋

## 前言
该游戏是基于Websocket开发的，采用Go + Bootstrap5   
你可以了解到:
- 五子棋盘的实现 --> internal/chessboard
- 简单的断线重连

### 特性
- [x] 创建房间、加入房间、离开房间
- [x] 随机先手
- [x] 游戏重开(仅房主)
- [x] 自动转移房主(对方退出房间)
- [x] 断线重连
- [x] 多人观战
- [x] 悔棋

### 开始
```
go run main.go
```
